Semantic HTML

Tag Name Description
<header> Header This is the content at the top of the page. It may include a logo, navigational links, or a search bar.
<nav> Nav This tag encapsulates the page's navigational links and is often placed in the header.
<main> Main This tag encapsulates the main content of the page between the header\navigation and the footer area.
<footer> Footer This section includes the page's footer content at the bottom of the body.
<section> Section This tag defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article This tag holds content that makes sense on it's own such as articles, blogs, and comments.
<aside> Aside This tag is used mark additional information that can enhance another element but isn't required in order to understand the main content.
<video> Video This tag allows us add videos to our website and requires a closing tag.
<audio> Audio This tag allows us to implement audio into our website and requires a closing tag.
<embed> Embed This tag can be used to implement any type of media and is self-closing.
<figure> Figure This tag is used to encapsulate media such as an image, diagram, or code snippet.
<figcaption> Figure Caption This tag is used to describe the media encapsulated within the figure.
